# ⚡ Circuito Lógico — Logisim

> Projeto de circuito lógico digital desenvolvido e simulado no Logisim-Evolution.

---

## 📋 Sobre o Projeto

Este projeto contém o arquivo de circuito lógico desenvolvido na disciplina de **Circuitos Digitais / Sistemas Digitais**. O circuito foi projetado, simulado e verificado utilizando o software **Logisim-Evolution**, uma ferramenta educacional de simulação de circuitos digitais.

---

## 🛠️ Requisitos

- **Software:** [Logisim-Evolution](https://github.com/logisim-evolution/logisim-evolution) (gratuito, requer Java)
- **Java:** JDK 11 ou superior

---

## 🚀 Como Abrir

1. Instale o Logisim-Evolution
2. Abra o arquivo `TRABALHO.circ` no Logisim:
   - `File > Open > TRABALHO.circ`
3. Use o modo de simulação para testar as entradas e verificar as saídas

---

## 📁 Estrutura

```
Cirucuito Logico - Logisim/
└── TRABALHO.circ    # Arquivo de circuito do Logisim
```

---

## 📌 Conceitos Aplicados

- Portas lógicas (AND, OR, NOT, NAND, NOR, XOR)
- Simplificação por Álgebra Booleana / Mapa de Karnaugh
- Simulação de sinais digitais
- Projeto de circuitos combinacionais e/ou sequenciais
